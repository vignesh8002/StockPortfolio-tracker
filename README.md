# StockPortfolio-tracker


A Python-based application designed to help users manage and monitor their stock investments efficiently. This project provides features for recording stock details, calculating investment values, tracking portfolio performance, and generating reports for better financial insights.

Features

- Add and manage stock information
- Calculate total investment value
- Track portfolio performance
- Generate portfolio reports
- Analyze profit and loss

Technologies Used

- Python
